# Impact Engine
__Impact Engine__ is currently in early access and under active development. Features and APIs may change as development continues.

**CHANGELOGS:**

- Added the solution generator (gsln.bat)
- Added premake
- Added loggers
- Implemented GLFW (And Window)
- Implemented Dear ImGui
- Implemented GLAD

**TUTORIAL**

Just double click gsln.bat, and the solution file will appear.

__NOTES__

- gsln.bat stands for Generate Solution.bat
- If you ever edit stuff in premake5.lua, make sure to double click gsln.bat, when Visual Studio ask you to reload the solution, click "Reload" or "Reload all".
